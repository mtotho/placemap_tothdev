####File Structure

api/    *REST API

api/application/controllers/<controllername.php>  *RESTful endpoints

api/application/models/<model_name.php> *Database logic

app/    *AngularJS App

app/controllers/ *Angular Controllers

res/    *images, css, misc

main.html   *template, app start point
